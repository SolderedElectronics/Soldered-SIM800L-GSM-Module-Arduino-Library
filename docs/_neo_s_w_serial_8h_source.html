26// This software serial library is intended as a more-efficient replacement
27// for SoftwareSerial at baud rates 9600, 19200 and 38400.
28//
29// Any of the pins supported by SoftwareSerial may be used. Pins (0-19)
30// on the Uno may be used. Other boards can use any of the pins
31// allowed by digitalPinToPCMSK in pins_arduino.h
32//
33// This code uses a pin change interrupt on the selected RX pin.
34// Transmission on the TX line is done in a loop with interrupts disabled.
35// Both RX and TX read timer0 for determining elapsed time. Timer0 itself is
36// not reprogrammed; it is assumed to be running with a 4 microsecond step.
37//
38// By default NeoSWSerial defines handlers for all PCINT interrupts like
39// SoftwareSerial. If client code requires own pin change interrupt handlers,
40// it's possible to rebuild library with #define NEOSWSERIAL_EXTERNAL_PCINT.
41// In such case client code should call NeoSWSerial::rxISR(PINB) (assuming
42// that receivePin is on PORT B)
43//
44// Supported baud rates are 9600 (default), 19200 and 38400.
45// The baud rate is selectable at run time.
46//
47// The size of the RX buffer may be changed by editing the
48// accompanying .cpp file. For optimal performance of the interrupt
49// service routines, the buffer size should be chosen to be a
50// power of 2 (i.e., 2, 4, 8, 16, 32, 64,...).
51//
52// v1.0 Nov 2014 jboyton - Created
53// v1.1 Jun 2015 jboyton - Added support for 8 MHz system clock. Timer 2 had to
54// be used since the timer 0 prescaler was inadequate
55// for this. The supported baud rates for 8 MHz are 9600
56// and 19200.
57// v2.0 Nov 2015 SlashDev - Add support for other boards,
58// add end() and attach/detachInterrupt
59// v2.1 Jun 2016 SlashDev - Add support for all character values
