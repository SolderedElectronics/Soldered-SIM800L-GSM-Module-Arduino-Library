1/* This is a bare bone library for communicating with SIM800
2 * It's barebone in that - it only provides basic functionalities while still
3 * maintaining strong performance and being memory friendly.
4 * It currently supports GSM(sending and reading SMS),
5 * GPRS connectivity(sending and receiving TCP) with Time and Location
6 *
7 * This library is written by Ayo Ayibiowu.
8 * charlesayibiowu@hotmail.com
9 * Designed to work with the GSM Sim800l module,
10 * and possibily other SIMxxx modules(not tested with others)
11 *
12 * This library use SoftwareSerial, you can define the RX and TX pin
13 * be sure that GND is attached to arduino too.
14 * The RESET_PIN is not used for this Library, you can leave it un-connected
15 *
16 * To Enable Debugging - Go to <BareBoneSim800.h file and change the
17 * #define DEBUG 0 to #define DEBUG 1
18 *
19 * PINOUT:
20 * _____________________________
21 * | ARDUINO UNO >>> SIM800L |
22 * -----------------------------
23 * GND >>> GND
24 * RX 8 >>> TX
25 * TX 9 >>> RX
26 *
27 * | ARDUINO MEGA >>> SIM800L |
28 * -----------------------------
29 * GND >>> GND
30 * RX 48 >>> TX
31 * TX 46 >>> RX
32 *
33 * POWER SOURCE 4.2V (MAX) >>> VCC
34 *
35 * Created on: Oct 24, 2017
36 * Author: Ayo Ayibiowu
37 * Email: charlesayibiowu@hotmail.com
38 * Version: v1.2
39 *
40 * ?????????????????????????????????????????????????
41 * Updates:
42 * SoftwareSerial upgraded to AltSoftSerial to support compartibility with other PinChange Interrupt Library
43 * and SPI based Library.
44 *
45 */
